# SeismoLab - 地震學互動教案

一個基於 HTML5、Tailwind CSS 與 SVG 動畫開發的地震學科普網頁。

## 特色
- **互動震波模擬**：利用 SVG 動畫呈現 P 波、S 波與表面波的物理特性。
- **響應式設計**：完美適配手機、平板與筆記型電腦螢幕。
- **教育功能**：包含逆推問題解析、震級對照表與冷戰核試驗監測應用。

## 如何部署
1. 將此專案 Fork 或 Clone 到你的 GitHub。
2. 進入 `Settings` > `Pages`。
3. Source 選擇 `Deploy from a branch`，並選擇 `main` 分支。
4. 儲存後，幾分鐘內即可透過 `https://你的帳號.github.io/你的專案名/` 瀏覽。
